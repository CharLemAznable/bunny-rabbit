服务计量模块.

#### 概述

使用```vert.x```框架开发, 实现高并发/非堵塞线程安全/轻量化, 支持嵌入server应用服务部署/单独服务部署, 支持应用服务内使用发布订阅模式调用/应用服务间使用REST接口模式调用.
